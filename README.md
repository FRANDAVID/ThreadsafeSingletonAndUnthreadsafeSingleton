# 深入理解单例模式及其多线程下的实现

本项目包含Java单例模式的各种经典实现，包括饥饿式实现和懒汉式实现。从线程安全角度看，本项目又包括6种线程安全实现：
 - 饿汉式
 -  私有静态内部类式
 -  synchronized方法式
 -  synchronized块式
 -  双重检查式
 -  ThreadLocal式
 
 和1种非线程安全实现：

 -  经典懒汉式。 
 
 此外，大家还可以参考我的博文[《彻头彻尾理解单例模式与多线程》](http://blog.csdn.net/justloveyou_/article/details/64127789)来深入了解单例模式的实现与原理。
